# Site Vitrine GRAD - TP d'introduction à PHP

Ce projet contient un site vitrine pour l'entreprise GRAD, utilisé comme support de cours pour un TP d'introduction à PHP.

## Contenu

- Site web statique (HTML/CSS) initial
- TP d'introduction à PHP (`TP_PHP_Introduction.md`)
- Toutes les ressources (images, CSS)

## Structure

- `page_d_accueil.html` - Page d'accueil
- `nos_prestations.html` - Page des prestations
- `nos_produits.html` - Page des produits
- `nos_realisations.html` - Page des réalisations
- `contact.html` - Formulaire de contact
- `site_ste_grad.css` - Feuille de style
- `TP_PHP_Introduction.md` - Document du TP

## Utilisation

Ce projet est destiné à être utilisé dans le cadre d'un cours d'introduction à PHP. Les étudiants doivent transformer progressivement le site statique en site dynamique en suivant les instructions du TP.

## Auteur

Projet pédagogique pour l'apprentissage de PHP.
